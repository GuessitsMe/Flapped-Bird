# Flappy Bird Game

## Overview
A browser-based Flappy Bird clone built with vanilla HTML, CSS, and JavaScript. The game runs entirely in the browser with no backend dependencies.

## Project Structure
- `index.html` - The complete game including HTML structure, CSS styles, and JavaScript game logic

## Features
- Classic Flappy Bird gameplay
- Day/night cycle based on score
- Animated clouds and buildings in background
- Pause functionality (P key)
- Score tracking with high score memory
- Optional leaderboard integration via Google Apps Script

## Running the Project
The game is served as a static website using Python's built-in HTTP server on port 5000.

## Controls
- **Space** or **Click** - Jump / Start game
- **P** - Pause/Resume

## Deployment
Configured for static deployment serving files from the root directory.
